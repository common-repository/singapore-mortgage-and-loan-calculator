Singapore Mortgage And Loan Calculator. A calculator plugin that helps you calculate monthly mortgage repayments for home loans in Singapore

== Description ==

Singapore Mortgage And Loan Calculator plugin is a service and tool for property and mortgage loans in Singapore. 
This WordPress plugin performs the mortgage calculation and loan eligibility test using the Loan Calculator API.

The plugin will call our service at http://www.executivecondominiums.com.sg/api/widget.js as the calculation is done on the server side. This way, we can update the needed parameter to calculate the loan (for example if the HDB interest rate is changed, etc.).
We guarantee that only data related to the calculation will be sent to our service, and no personal data will be sent.

Basic functions of this calculator-
It is able to do basic mortgage calculation for:
1) Singapore HDB Loans
2) Private Condominiums mortgage loans in Singapore

Advance functions of this calculator-
The calculator can determine the loan eligibility of the applicant, depending on the applicant's:
1) Age
2) Loan amount relative to property purchase price
3) Loan tenure
4) Loan number (Number of outstanding private property loans)

Suitable for property websites and loan comparison websites.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to Appearance->Widgets to find the Plugin under Available Widgets
1. Add the widget to your desired area


== Frequently Asked Questions ==

= Is this plugin specifc to Singapore only? =

No. The calculator consists of 3 separate tabs. HDB Loan calculator tab, Bank Loan calculator tab and Loan Eligibility calculator tab.
If your site audience is not primary for Singapore, you can make use of the Bank Loan calculator which is a generic home mortgage calculator.
If your site is for Singapore visitors, the HDB Loan calculator has Interest Rate fixed at 2.6%, and the Loan Eligibility calculator helps to determine the loan eligibility of the applicant based on their age and number of oustanding loans.
